# Introduction
This repo for Helper services.

# Installation

### via SSH
```git
git clone git@bitbucket.org:loyaltoid/go-skeleton-manager.git
```
### via go modules
```git
go mod install github.com/Qoin-Digital-Indonesia/qoin-go-helper.git
```

# Usage
```shell
qoinhelper.LoggerInfo("This is logger info")
```
